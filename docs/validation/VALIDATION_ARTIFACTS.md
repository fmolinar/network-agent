# Validation Artifacts

## Test Suite
Run:

```bash
pytest -q
```

Representative validated behaviors:
- Planner categorization and platform mapping.
- Executor live collection behavior and Windows ping command bounds.
- Packet-capture duration behavior (`capture_seconds` and prompt-based override).
- Validator safety gating and command blocking.
- Engine-level orchestration and debug trace.
- LLM integration behavior via mock provider.

## Example Constraint Checks

### Safety / command controls
- Forbidden commands are blocked (e.g., `rm -rf /`).
- Dual-use mutation commands require explicit approval.

### Data validity
- Diagnosis confidence must remain in `[0,1]`.
- Parsed packet loss must remain in `[0,100]`.
- Missing cause lists are rejected.

### Capture guardrails
- Live packet capture duration is bounded.
- Timeout behavior returns partial output instead of hard crashing.
